GBA Soccer

C

View Project

        GBA Soccer is a multi-level soccer game for the Game Boy Advance built in C. The game relies on several fundamental low-level programming concepts such as the usage of a state machine to manage game state and display, accessing GBA components such as the screen, buttons, and DMA controller through memory-mapped I/O, as well as bit-level operations and optimization techniques to set register values and eliminate tearing.

Pokemon Hangman

JavaScript, React, Node, Express, MongoDB, HTML/CSS

View Project
